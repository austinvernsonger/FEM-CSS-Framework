FEM CSS Framework

The base is a 12 columns grid. The total width is 960. The columns have 10px margin in each side, so it makes 20px gutter between columns.

We can remove that margin or expand or contract the box to play with that gutter.

The idea is to wrap each group of boxes inside another, so they won't mix with the next floated boxes and also we can decorate with borders or backgrounds
